Build a complete, production-ready landing page from scratch using v0. Each section has a tested prompt.

Hero Section

Centered layout, gradient headline, dual CTAs, trust badge logos, dark theme. Generate this first to set the tone.

Features Section

Three-column grid, icon per card, subtle borders, hover animations, light background.

Social Proof

Three testimonial cards with avatar, name, role, quote, star rating. "Trusted by" headline above.

Pricing Cards

Three tiers with feature checklists, highlighted middle tier with "Most Popular" badge, monthly/annual toggle.

FAQ Section

Six expandable questions in accordion format with plus/minus icons.

Final CTA

Full-width section with bold headline, supporting text, large button, gradient background.

Review and Deploy

Use Design Mode to adjust spacing and colors. Check mobile preview. Click "Deploy to Vercel."

Pro Tip

Generate each section separately rather than asking for the entire page in one prompt. Section-by-section gives you more control over quality.

Total time: 20-30 minutes for a professional, deployed landing page.
