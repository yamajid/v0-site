Security & Compliance

Enterprise plans include enhanced security features required by regulated industries.

Compliance Certifications

  • SOC 2 Type II: Verified security controls and processes
  • GDPR Compliance: Data processing agreements available
  • HIPAA: BAA available for healthcare organizations
  • CCPA: California privacy requirements supported

Data Security

  • Data residency: Choose where your data is stored
  • Encryption: AES-256 at rest, TLS 1.3 in transit
  • Data retention: Configurable retention policies
  • Audit logs: Complete activity logging for compliance

Security Review

Enterprise customers can request Vercel's security documentation, penetration test results, and complete a vendor security questionnaire during evaluation.

Access Controls

  • IP allowlisting: Restrict access to approved networks
  • Session management: Configure session duration and policies
  • MFA enforcement: Require multi-factor authentication
  • Activity monitoring: Real-time security alerts

Team Management

Manage large teams with granular controls.

Team Features

Onboarding
