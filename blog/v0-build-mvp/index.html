The MVP Project

A SaaS task management app with landing page, user dashboard, task creation/management, team invites, and settings.

Results

Total time: ~48 minutes | Credits: ~$4.30 | Pages built: 6 | Quality: Production-ready UI

Landing Page (5 min, ~$0.50)

Hero with CTA, feature grid, testimonials, pricing tiers. Used the three-part prompt framework.

Auth UI (8 min, ~$0.80)

Login, signup, forgot password with form validation and OAuth button.

Dashboard (12 min, ~$1.50)

Sidebar navigation, task table with priority badges, Add Task modal, stat cards. Used agentic mode for complexity.

Task Detail (8 min, ~$1.00)

Edit form, assignee dropdown, subtasks checklist, comments, action buttons.

Settings (5 min, ~$0.50)

Tabbed layout: Profile, Notifications, Team, Billing, Danger Zone.

Connect and Deploy (10 min, $0)

Export to GitHub, connect to Vercel, add environment variables, set up Supabase.

Key Lessons

  • Start with landing page to define your product clearly
  • Build core workflow first: create task, view dashboard, manage task
  • Use agentic mode for complex components
  • Export to GitHub early
  • Deploy incrementally — get each page live before building the next
