v0 by Vercel lets you describe a UI in plain English and get back production-ready React code — with a live visual preview — in seconds. Whether you want a landing page, a dashboard, a pricing table, or a full multi-page app, v0 generates it using React, Tailwind CSS, and shadcn/ui.

This guide covers everything from creating your account to deploying your first live component. The whole process takes about 15 minutes. You will finish with something real, shareable, and running on HTTPS.

What Is v0, Exactly?

v0 is an AI-powered UI generation tool built by the Vercel team — the same people behind Next.js. You type what you want, and v0 streams a complete React component into a split-screen view: code on the left, live interactive preview on the right. It is not just a code autocomplete tool. The preview is fully interactive before you touch a single line of code.

Under the hood, v0 generates components using:

  • shadcn/ui — open-source component library with accessible, beautiful defaults
  • Tailwind CSS — utility-first CSS that makes components responsive automatically
  • Next.js / React — the framework stack built for production

You can generate a single component (a button group, a form, a modal), a full page (a SaaS hero, an auth screen, a dashboard), or an entire application with multiple routes. Read our full explainer on what v0 is for more context on where it fits in the modern builder stack.

Before You Start: What You Need

Getting started with v0 requires almost nothing:

  • A Vercel account or GitHub account — both are free
  • A browser — no local installation required
  • No coding knowledge required to generate; basic React knowledge helps you integrate the output

v0's free tier includes $5 in monthly credits with no credit card required. That is enough to generate dozens of components before you need to think about upgrading. See our complete v0 pricing breakdown to understand exactly what each plan includes.

Step 1: Create Your Account

Go to v0.dev. Click Sign Up and authenticate with GitHub (recommended if you plan to sync code to a repo) or with your Vercel account (recommended if you plan to deploy directly). Email sign-up also works but has fewer integration features.

Once signed in you land on the v0 home screen: a clean interface with a chat prompt input at the center and example generations below it. Your free $5 monthly credits are automatically applied. No setup, no configuration, no local dependencies to install.

One important note: your v0 account is tied to a Vercel workspace. If you are part of a team, confirm you are signed into the right account before generating — projects you create will appear in that workspace.

Step 2: Write Your First Prompt

This is where most beginners either get great results immediately or struggle. The key insight: v0 responds best to specific, visual descriptions. Think like a designer briefing a developer — describe what you see, not what you want to achieve.

Here is a strong beginner prompt to copy and try:

"Create a SaaS landing page hero section with a bold gradient headline, a one-line subheadline, two CTA buttons (primary green and secondary ghost), and a row of three trust badges. Dark background. Modern, minimal."

What makes this prompt work: it names the component type (hero section), lists every specific element (headline, subheadline, 2 CTAs, 3 badges), specifies the visual style (dark, minimal), and even specifies button variants. Compare that to a bad prompt like "make a landing page" — too vague to get a useful result.

After you hit enter, v0 streams the generation in real time. Code appears on the left as the preview updates on the right. This usually takes 10–30 seconds for a component and up to 60 seconds for a full page. See our complete v0 prompting guide for advanced techniques and a library of ready-to-use prompts.

Step 3: Preview, Interact, and Iterate

When generation finishes, the right panel renders your component in a browser-like preview. You can click buttons, fill in form fields, hover over elements — it is fully interactive. You can also:

  • Drag the preview panel's edge to check how it looks at different widths (mobile responsiveness)
  • Toggle dark/light mode to see both themes
  • Switch between the Code, Preview, and Console tabs
  • Ask for changes by typing a follow-up prompt in the chat

Follow-up prompts that consistently work well:

✅ Effective follow-up prompts:

  • "Make the headline font size larger and add a text gradient from green to blue"
  • "Replace the placeholder trust badges with logos: Stripe, Notion, Linear"
  • "Add a subtle animated mesh gradient background"
  • "Make the layout stack vertically on mobile"
  • "Add a navigation bar above the hero with logo and three nav links"

Each prompt costs a small fraction of your credits. Simple changes (color, text, spacing) cost less than generating new sections. For zero-credit edits, use Design Mode.

Step 4: Use Design Mode — It Is Free

Design Mode is one of the most useful and most overlooked features for beginners. Click the paintbrush icon in the top toolbar to switch from AI chat to visual editing. In Design Mode you can:

  • Click any text element to edit the copy directly — no prompt needed
  • Select elements to adjust padding, margin, font size, and color in a sidebar panel
  • Drag elements to reposition them
  • Replace placeholder images with your own URLs

Design Mode consumes zero credits. It is pure visual editing, no AI generation involved. This is the right tool for replacing placeholder text with your real copy, tweaking colors to match your brand, and making small layout fixes. For deeper features, read our full Design Mode tutorial.

Step 5: Export Your Code

When you are satisfied with the result, you have four export paths depending on your workflow:

Copy Code — Click the Code tab on the left, select all, and copy. Paste the JSX into your existing Next.js or React project. Best for quick prototyping or when you already have a codebase.

Export as ZIP — Click Export and v0 packages a complete Next.js project: package.json, all dependencies installed, components organized by folder, Tailwind config included. Download it, unzip, run npm install && npm run dev and you are running locally.

GitHub Sync — Connect v0 to a GitHub repository. Changes sync automatically. Every time you refine your component in v0, the repo updates — and if your repo is connected to Vercel, a deployment triggers automatically. This is the most powerful workflow for ongoing projects. Our GitHub integration guide walks through the full setup.

Add to Existing v0 Project — If you already have a v0 project from a previous session, you can add the new component directly to that project's file tree rather than starting a fresh project.

Step 6: Deploy to Vercel

If you used the ZIP export or GitHub sync, deployment is a single click. Hit the Deploy button inside v0's interface. You will be prompted to choose or create a Vercel project, confirm the build settings (the framework is auto-detected as Next.js), and click Deploy.

Vercel runs the build and returns a live HTTPS URL — typically in 30 to 90 seconds. The URL is immediately shareable. Every subsequent push via GitHub sync triggers an automatic rebuild. You get a free .vercel.app subdomain instantly, and you can attach a custom domain from the Vercel dashboard at any time. For full deployment details including environment variables and domain setup, read our v0 deployment guide.

Pro Tips for Beginners

Start with one component, not an entire page. Generating a hero section or a pricing table is easier to iterate on than a full multi-section page. Build components one at a time, then ask v0 to compose them into a page layout.

Attach a reference image. You can include an image in your prompt by pasting a URL or uploading a screenshot. Showing v0 a design you want to replicate is often more effective than describing it. This works especially well for matching a competitor's layout or recreating a Dribbble design.

Use v0 to learn React. After generating a component, type "explain what each part of this component does" in the chat. v0 will walk through the code section by section. This is a surprisingly effective way to learn modern React patterns.

Save your best prompts. v0 does not yet have a built-in prompt library. When a prompt produces an excellent result, paste it into a text file for reuse. See our curated library of v0 landing page prompts.

Iterate, do not regenerate. When something is mostly right but needs fixing, use a follow-up prompt rather than starting over. Targeted refinements are more efficient and produce more consistent results than regenerating from scratch.
