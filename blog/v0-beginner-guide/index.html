v0 (at v0.app) is Vercel's AI-powered app builder. You describe what you want in plain language, and v0 generates real, working code. Think of it as a senior developer on call who builds UI from your descriptions.

What Is v0?

Originally launched as v0.dev, Vercel's AI platform has evolved into v0.app — a complete agentic AI builder. Instead of just generating code snippets, v0 now plans, reasons, searches the web, inspects live sites, and fixes its own errors autonomously.

Under the hood, v0 generates React components with Next.js, Tailwind CSS, and shadcn/ui — deploying to Vercel's global edge network with a single click.

Key Technology Stack

v0 generates code using Next.js, React, TypeScript, Tailwind CSS, and shadcn/ui components — the modern web development stack trusted by companies worldwide.

Getting Started in 5 Steps

  1. Go to v0.app — sign in with Vercel or GitHub, land on a blank canvas with $5 in free credits
  2. Describe what you want — be specific about what you're building, who uses it, and how it should look
  3. Watch v0 generate the UI in real time on the preview panel
  4. Iterate with follow-ups — refine details via additional prompts or Design Mode
  5. Deploy or export — push to Vercel with one click, or sync to GitHub

What Can v0 Build?

Great at: Landing pages, dashboards, admin panels, authentication UI, e-commerce pages, data tables, forms, navigation systems, portfolio sites, internal tools, and multi-page applications.

Limited at: Complex backend logic, database migrations, full mobile app development. For these, integrate with external services after the UI is built.

Pricing Overview

v0 uses a credit-based model. The Free plan gives you $5 in monthly credits. Premium at $20/mo unlocks 4x more credits, the v0-1.5-lg model, and Figma import. Team at $30/user/mo adds shared workspaces and centralized billing.

Your First Prompt

Try this to start:

"Build a personal portfolio landing page with a hero section showing my name and role, a projects grid with 3 cards, an about section, and a contact form at the bottom. Use a dark theme with blue accent colors."

The more specific your prompt, the better the output. Always include what you're building, who uses it, and how it should look.
