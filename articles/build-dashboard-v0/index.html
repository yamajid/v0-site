Building a Dashboard with v0 — Step by Step

Learn how to create a complete analytics dashboard using v0. This tutorial covers stat cards, charts, data tables, and sidebar navigation.

What You'll Build

A fully functional analytics dashboard with:

  • Sidebar navigation with icons
  • Top header with user profile
  • 4 stat cards (users, revenue, orders, growth)
  • Line chart for trends
  • Data table with sorting
  • Recent activity feed

Step 1: Sidebar Navigation

Start by creating the sidebar:

Prompt

Create a dashboard sidebar with: - Fixed width 250px, dark background (#0a0a0a) - Logo at top - Navigation items: Dashboard, Analytics, Customers, Products, Settings - Each item has an icon (Lucide) - Active state with emerald highlight - Hover effect on items - User profile section at bottom - Collapsible on mobile with hamburger menu

Step 2: Stat Cards

Create the KPI stat cards:

Prompt

Create a row of 4 stat cards: - "Total Users" with count and +12% change - "Revenue" with dollar amount and +8% change - "Orders" with count and +23% change - "Growth" with percentage and +5% change - Each card: icon, label, value, change indicator (green up arrow) - Dark card background, subtle border - Hover lift effect

Step 3: Charts Section

Add analytics charts:

Prompt

Create a charts section with: - Two charts side by side on desktop, stacked on mobile - Chart 1: Line chart showing "Revenue Over Time" - 12 months data - Chart 2: Bar chart showing "Users by Source" - Organic, Paid, Referral, Direct - Dark theme with emerald accent colors - Titles and legends - Tooltips on hover

Step 4: Data Table

Build the data table:

Prompt

Create a data table with: - Columns: Customer, Email, Plan, Status, Amount, Date - 10 sample rows of data - Sortable headers (click to sort) - Row hover effect - Status badges (Active=green, Pending=yellow, Cancelled=red) - Pagination at bottom - Search input above table - Dark theme styling

Step 5: Activity Feed

Add a recent activity section:

Prompt

Create an activity feed with: - Title "Recent Activity" - List of 8 recent activities - Each item: icon, description, relative time ("2 hours ago") - Timestamps in lighter color - Scroll if content overflows - Dark theme

Full Dashboard Prompt

Once comfortable, combine everything:

Complete Dashboard Prompt

Build a complete admin dashboard with: 1. Sidebar: Logo, nav items (Dashboard, Analytics, Customers, Products, Settings) with icons, user profile at bottom 2. Main content area with header showing "Welcome back" and user avatar 3. 4 stat cards in a row: Users, Revenue, Orders, Growth with icons and % change 4. Two charts: Line chart (revenue) and Bar chart (users by source) 5. Recent orders table with columns (Customer, Product, Amount, Status, Date) 6. Activity feed sidebar on right Use: - Dark theme (#030303 bg) - Emerald accent (#10b981) - Inter font - Tailwind CSS - Fully responsive - Smooth hover animations
