How to Build a SaaS Landing Page with v0

Learn how to create a professional SaaS landing page using Vercel v0. This step-by-step guide covers everything from crafting effective prompts to deploying your finished page.

What You'll Build

By the end of this tutorial, you'll have a complete SaaS landing page featuring:

  • Hero section with headline, subtext, and CTA
  • Features grid showcasing your product benefits
  • Pricing table with tier options
  • Testimonials section
  • FAQ accordion
  • Footer with links and social icons

Step 1: Create the Hero Section

The hero section is the first thing visitors see. Use this prompt to generate a compelling hero:

Prompt

Create a modern SaaS hero section with: - Bold headline text promoting an AI coding assistant - Subheadline explaining the value proposition - Two CTA buttons: "Start Free" (primary emerald) and "View Demo" (outline) - Animated gradient background effect - Floating UI mockup illustration on the right side - Use Inter font, dark theme with emerald accent colors - Include trust badges showing "10,000+ developers" and "4.9 rating" - Make it fully responsive with mobile optimization

What v0 Will Generate

v0 will create a hero section with:

  • Responsive layout (stacks on mobile)
  • Gradient animation effects
  • Proper button styling with hover states
  • Trust indicators with icons

Step 2: Add Features Section

Now let's create a features grid to showcase your product's benefits:

Prompt

Create a SaaS features section with: - 3x2 grid of feature cards - Each card has an icon, title, and description - Dark card background (#0a0a0a) with subtle border - Hover effect: slight lift and border color change to emerald - Icons should be Lucide React icons - Features: "AI-Powered", "Lightning Fast", "Secure", "Team Collaboration", "Analytics", "24/7 Support" - Section headline: "Everything you need to build better software"

Step 3: Build Pricing Table

A well-designed pricing table can significantly boost conversions:

Prompt

Create a pricing section with: - 3 pricing tiers: Starter ($0), Pro ($29), Enterprise ($99) - Pro tier highlighted with "Most Popular" badge and emerald border - Each tier shows: name, price, list of features, CTA button - Monthly/yearly toggle at the top (yearly shows 20% discount) - Checkmark icons for features - Dark theme, rounded cards, smooth hover animations - Responsive: stacks on mobile, 3-column on desktop

Step 4: Add Testimonials

Social proof builds trust. Create a testimonials section:

Prompt

Create a testimonials section with: - 3 testimonial cards in a row - Each card shows: quote text, user avatar, name, role, company - Star rating (5 stars) above each quote - Subtle card background with border - Quote icon decoration - Real-sounding testimonials about improving developer productivity - Avatar placeholder images using ui-avatars.com

Step 5: FAQ Accordion

Address common questions with an interactive accordion:

Prompt

Create an FAQ accordion section with: - 5 common SaaS questions with answers - Click to expand/collapse functionality - Plus/minus icon animation on expand - Smooth height transition - Questions: "Is it free?", "How does billing work?", "Can I cancel anytime?", "What support is included?", "Is my data secure?" - Clean dark theme styling

Step 6: Professional Footer

Complete your landing page with a comprehensive footer:

Prompt

Create a SaaS footer with: - 4 columns: Product, Resources, Company, Legal - Company logo and tagline in first column - Social media icons (Twitter, GitHub, LinkedIn) - Copyright text - Dark background matching theme - Newsletter signup input field - All links styled consistently

Full Page Assembly Prompt

Once you've tested individual sections, combine them all at once:

Complete Landing Page Prompt

Create a complete SaaS landing page with all these sections in order: 1. Navbar: Logo, nav links (Features, Pricing, About, Blog), "Get Started" button 2. Hero: Headline "Build Faster with AI", subtitle, 2 CTAs, animated gradient bg 3. Logos: "Trusted by" section with company logos (placeholder) 4. Features: 3x2 grid with icons, titles, descriptions 5. Pricing: 3-tier table with toggle, highlighted middle tier 6. Testimonials: 3 cards with quotes, avatars, names 7. CTA: Final call-to-action with email input 8. Footer: 4-column links, social icons, copyright Use: - Dark theme (#030303 bg) - Emerald accent (#10b981) - Inter font - Tailwind CSS with shadcn/ui components - Fully responsive design - Smooth animations and hover effects

Deployment

Once your landing page is ready in v0:

Step 1: Connect GitHub

Click the GitHub icon in the v0 editor to connect your repository. v0 will create a new repo or push to an existing one.

Step 2: Deploy to Vercel

v0 automatically deploys to Vercel. You'll get a live URL like your-project.vercel.app within seconds.

Step 3: Custom Domain (Optional)

In Vercel dashboard, go to Settings → Domains to add your custom domain. DNS propagation takes 24-48 hours.

Pro Tips

  • Iterate gradually: Build section by section rather than all at once for better results
  • Be specific: The more detail you provide about colors, fonts, and behavior, the better the output
  • Use reference sites: Mention specific websites as style references
  • Test on mobile: Always check how sections look on mobile before moving on
