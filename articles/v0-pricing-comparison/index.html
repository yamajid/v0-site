What Are Credits?

Credits are used each time you interact with v0:

  • Generation: ~10-50 credits per component
  • Edit: ~5-15 credits per edit
  • Preview: ~1-5 credits per preview

A typical component takes about 20 credits to generate. With 200 free credits, you can create ~10 components per month.

Who Should Stay on Free?

  • Learning v0 for the first time
  • Building personal projects
  • Testing before committing
  • Creating ~10 or fewer components/month

Who Should Upgrade to Premium?

  • Building client projects
  • Working on commercial products
  • Need private repositories
  • Working in a team
  • Creating 50+ components/month
  • Need priority support
