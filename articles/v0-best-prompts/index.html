50 Best v0 Prompts for Developers

A curated list of 50 tested prompts for building common UI components, layouts, and features with Vercel v0.

🎨 Hero Sections

1. Modern SaaS Hero

Create a SaaS hero section with bold headline, subtext, two CTA buttons, animated gradient background, and floating product mockup. Use dark theme with emerald accents. Make it fully responsive.

2. Minimal Hero

Create a minimal hero with centered headline, short description, single primary CTA, and subtle background pattern. Use lots of whitespace and clean typography. Dark theme.

3. Hero with Stats

Build a hero section with headline, value proposition, CTA, and 3 stats below (users count, rating, projects). Include company logos of trusted companies. Modern dark design.

🃏 Cards & Grids

4. Feature Cards Grid

Create a 3x2 grid of feature cards. Each card has an icon, title, and description. Dark background cards with subtle border, lift on hover with emerald border glow.

5. Bento Grid Layout

Build a modern bento grid with cards of varying sizes. Include one large card (2x2), two medium cards (2x1), and four small cards. Dark theme, rounded corners, hover effects.

6. Testimonial Cards

Create 3 testimonial cards in a row. Each has quote text, 5-star rating, user avatar, name, role, and company. Dark cards with subtle border. Responsive grid.

💰 Pricing

7. Pricing Table

Build a pricing table with 3 tiers: Starter ($0), Pro ($29), Enterprise ($99). Include monthly/yearly toggle. Highlight middle tier. Feature lists with checkmarks. Dark theme.

8. Pricing with Comparison

Create a pricing comparison table with 3 plans. Show features as checkmarks/crosses. Highlight recommended plan. Include FAQ accordion below. Clean dark design.

📝 Forms

9. Login Form

Build a login form with email and password fields, "Remember me" checkbox, submit button, "Forgot password?" link, and "Sign up" link. Include validation states. Dark themed.

10. Contact Form

Create a contact form with name, email, subject dropdown, message textarea, file upload, and submit button. Show loading state. Dark theme with emerald accent button.

11. Multi-Step Form

Build a 3-step form wizard: Step 1 (Personal Info), Step 2 (Preferences), Step 3 (Review). Show progress indicator. Dark theme, smooth transitions between steps.

12. Newsletter Signup

Create a newsletter signup with email input, subscribe button, privacy note, and success state. Inline design. Dark theme with emerald button. Animate on submit.

🧭 Navigation

13. Navbar with Dropdown

Build a responsive navbar with logo, nav links, and CTA button. Add dropdown menus on hover for "Products" and "Resources". Mobile hamburger menu. Sticky on scroll.

14. Mobile Navigation

Create a mobile bottom navigation with 5 icons: Home, Search, Add, Notifications, Profile. Fixed at bottom. Active state indicator. Dark theme.

15. Breadcrumbs

Build breadcrumbs navigation showing current page path: Home / Products / Category / Item. Clickable links, current page not linked. Dark theme styling.

👤 Dashboard

16. Stats Cards

Create 4 stat cards in a row: Total Users, Revenue, Orders, Growth. Each shows label, value, percentage change with up/down arrow. Dark cards with icons.

17. Data Table

Build a data table with 5 columns: Name, Email, Status, Role, Actions. Include 10 rows of sample data. Sortable headers, row hover effect. Dark theme.

18. Sidebar Navigation

Create a sidebar with logo at top, nav items with icons, section dividers, and user profile at bottom. Collapsible on mobile. Dark background.

📦 Components

19. Modal Dialog

Build a modal with header (title + close X), body content, and footer (cancel + confirm buttons). Backdrop blur. Dark theme. Animate in/out.

20. Toast Notifications

Create toast notification component with success, error, warning, and info variants. Show icon, message, and close button. Stack from top-right. Dark theme.

21. Accordion FAQ

Build an accordion with 5 FAQ items. Click to expand/collapse. Show plus/minus icon. Smooth height animation. Dark theme.

22. Tabs Component

Create tab navigation with 3 tabs: Overview, Features, Pricing. Underline active tab. Tab content area. Dark theme.

🛒 E-commerce

23. Product Card

Build a product card with image, title, price, original price (struck), rating stars, "Add to Cart" button. Hover effect with shadow. Dark theme.

24. Product Detail Page

Create a product page with image gallery (thumbnails), title, price, description, size selector, quantity picker, and "Add to Cart" button. Dark theme.

25. Shopping Cart

Build a cart sidebar with item list (image, title, qty, price), remove button, subtotal, and checkout button. Slide in from right. Dark theme.

📱 Blog

26. Blog Card

Create a blog post card with featured image, category tag, title, excerpt, author avatar, name, and date. Hover lift effect. Dark theme.

27. Blog Post Layout

Build a blog post layout with hero image, title, author info, publish date, content area with headings, and sidebar with related posts. Dark theme.

🦸 Buttons & CTAs

28. Button Variants

Create all button variants: primary (emerald), secondary (outline), ghost, danger. Include small, medium, large sizes. Show hover and active states.

29. Call to Action Section

Build a CTA section with headline, subtext, email input, and submit button. Background gradient. Centered content. Dark theme.

🔻 Footer

30. Multi-Column Footer

Create a footer with 4 columns: logo+tagline, Product links, Company links, Legal links. Newsletter signup. Social icons. Dark theme.

31. Minimal Footer

Build a minimal footer with logo, copyright text, and social links. Centered. Clean dark theme.

🎭 Advanced

32. Kanban Board

Create a kanban board with 3 columns: To Do, In Progress, Done. Drag and drop cards. Each card has title and priority tag. Dark theme.

33. Calendar View

Build a calendar component showing current month. Days with events highlighted. Navigation arrows for month. Dark theme.

34. Settings Page

Create a settings page with sidebar navigation and content area. Include sections: Profile, Account, Notifications, Security. Dark theme.

35. Chat Interface

Build a chat interface with message list, input field, send button. Show user and AI messages differently. Typing indicator. Dark theme.

🔐 Auth

36. Sign Up Form

Create signup form with name, email, password, confirm password, terms checkbox, and submit button. Show password strength. Dark theme.

37. Forgot Password

Build a forgot password page with email input and submit button. Show success message after submit. Dark theme.

38. Profile Settings

Create profile settings with avatar upload, name, email, bio, and save button. Show current avatar. Dark theme.

📊 Charts

39. Line Chart

Build a line chart showing 30-day trend. X-axis dates, Y-axis values. Tooltip on hover. Dark theme with emerald line.

40. Bar Chart

Create a bar chart comparing 5 items. Horizontal bars with labels. Tooltip on hover. Dark theme.

41. Donut Chart

Build a donut chart with 4 segments. Legend below. Center shows total. Dark theme.

🖼️ Media

42. Image Gallery

Create an image gallery with 6 thumbnails in grid. Click to view full size in modal. Arrow navigation. Dark theme.

43. Video Player

Build a video player with custom controls: play/pause, progress bar, volume, fullscreen. Dark theme styling.

🎯 Micro-interactions

44. Loading Spinner

Create an animated loading spinner. Emerald color. Multiple variations: small, medium, large.

45. Skeleton Loading

Build skeleton loaders for: text lines, image placeholder, card. Animated shimmer effect. Dark theme.

46. Progress Bar

Create a progress bar with percentage label. Animated fill. Dark theme with emerald fill color.

🔔 Alerts

47. Alert Banner

Build alert banners: success (green), error (red), warning (yellow), info (blue). Include icon, message, and close button. Dark theme.

48. Badge/Chip

Create badge components: status badges (success, warning, error), notification count, category tags. Various colors. Small size.

🏢 Pages

49. 404 Page

Build a 404 error page with "Page Not Found" message, illustration, "Go Home" button. Creative dark design.

50. Coming Soon Page

Create a coming soon page with headline, description, countdown timer, email signup, and social links. Dark theme.
